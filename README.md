# ☕ CafePulse: Transaction & Revenue Insights


This project analyzes ~8.5K cafe transactions to uncover insights on revenue, product performance, and customer behavior using SQL and Power BI.

---

## 🛠 Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard)

---

## 📊 Key Insights
- Salad is the top-performing item (~90K revenue, highest quantity)
- Tea is the least performing item
- Juice and Smoothie have similar revenue but different quantity trends
- Sales peak in January and drop in later months
- Most used payment mode is "Other", followed by Credit Card
- In-store sales are the second highest revenue source
